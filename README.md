IMDb Data Analysis

Analysis of IMDb and streaming content data to understand trends in movies and TV shows using data cleaning and interactive visualization techniques.

Project Overview

--- Analysis of movie and TV show content trends
--- Exploration of genre, language, and release patterns
--- Comparison of movie and series distribution
--- Duration and content growth analysis

Dataset

--- Dataset contains ~9000 titles including movies and TV shows
--- Attributes include genres, language, release year, and duration
--- Dataset source: Kaggle Netflix Shows Information dataset

Tools Used

--- Python (Pandas, NumPy) for data analysis
--- Jupyter Notebook for data cleaning
--- Power BI for dashboard visualization

Analysis Performed

--- Cleaning and preprocessing raw dataset
--- Genre and language distribution study
--- Content type comparison (Movies vs Series)
--- Release trend analysis over time
--- Duration pattern analysis
